Save2Drive

Save2Drive is a simple Chrome extension that saves links, images, HTML5 audio and videos directly from Chrome to Google Drive. Install from Chrome Web Store, or fork the GitHub repo.

Save2Drive was featured on Lifehacker, Addictive Tips, CNET, Free Technology for Teachers, Gigazine, GuAo and other online media. I feel thrilled to see the blogs that I have subscribed to since high school are posting about my work. The write-up from Addictive Tips is especially elaborated and detailed. Thank you for all the support!

The hack behind Save2Drive is fairly easy. Google Docs provides an online viewer, and some URL parameter tricks will make the viewer save a file with just a link (to a file, an image, or anything). Simplicity brought along limitations. This is not a documented feature of Google Docs, and I was not able to find out more flexibility to add to this extension. Many users complained that they were not able to preview images saved via Save2Drive, or all the files go to the root folder instead of a user-selected one. Sorry but there is nothing I can do at the moment, unless I integrate Save2Drive with Google Drive API.
